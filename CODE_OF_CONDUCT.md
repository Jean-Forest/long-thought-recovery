# Code of Conduct

이 프로젝트는 회복과 자기관리 도구를 만드는 공간입니다. 모든 참여자는 서로를 존중해야 합니다.

## 기대하는 행동

- 친절하고 명확하게 말하기
- 사용자의 어려움을 조롱하지 않기
- 의학적 주장을 과장하지 않기
- 근거 없는 치료 효과를 홍보하지 않기
- 다양한 생활 환경과 집중 어려움을 존중하기

## 허용하지 않는 행동

- 모욕, 괴롭힘, 차별적 발언
- 개인정보 공개
- 특정 사용자의 건강 상태에 대한 단정
- 검증되지 않은 치료법 홍보

## 문제 제기

문제가 생기면 repository 관리자에게 알려주세요.
