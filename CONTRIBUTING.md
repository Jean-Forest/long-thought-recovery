# Contributing

기여를 환영합니다. 이 프로젝트는 짧은 자극에 익숙해진 사람들이 긴 생각과 인지적 인내력을 회복하도록 돕는 도구를 목표로 합니다.

## 기여할 수 있는 영역

- UI/UX 개선
- 모바일 사용성 개선
- 접근성 개선
- 훈련 프롬프트 추가
- 계산/시각화 과제 개선
- 그래프와 리포트 개선
- 문서화
- 버그 수정

## 원칙

1. 사용자를 탓하지 않습니다.
2. 의학적 효과를 과장하지 않습니다.
3. 개인정보를 서버로 보내지 않는 방향을 우선합니다.
4. 완벽주의보다 지속 가능한 회복 루틴을 우선합니다.
5. 어려운 날을 위한 최소 루틴을 중요하게 다룹니다.

## 개발 방식

현재 앱은 빌드 과정 없는 정적 웹앱입니다.

```bash
# 로컬에서 간단히 확인
python3 -m http.server 8000
```

그 다음 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8000
```

## Pull Request 전 확인

- [ ] `index.html`이 정상적으로 열리는가?
- [ ] localStorage 저장이 동작하는가?
- [ ] JSON 백업/복원이 동작하는가?
- [ ] 모바일 폭에서 레이아웃이 깨지지 않는가?
- [ ] 의료적 효과를 단정하는 문구가 없는가?
