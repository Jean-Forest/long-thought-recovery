# Privacy

Long Thought Recovery는 기본적으로 서버가 없는 정적 웹앱입니다.

## 수집하는 데이터

앱은 사용자가 직접 입력한 다음 정보를 브라우저 localStorage에 저장합니다.

- 하루 체크 기록
- 쇼츠 사용 시간
- 수면 시간
- 운동 여부
- Long Thought, 계산, 시각화, 독서 기록
- 머리 맑음, 집중감, 시각화 선명도 점수
- 저널과 메모
- 시작일 설정

## 서버 전송

현재 버전은 데이터를 외부 서버로 전송하지 않습니다.

## 사용자가 해야 할 일

- 공용 컴퓨터에서는 사용하지 않는 것이 좋습니다.
- 기기를 바꾸기 전 JSON 백업을 저장하세요.
- 민감한 개인정보나 타인의 개인정보는 저널에 쓰지 않는 것이 좋습니다.

## 향후 서버 기능이 추가될 경우

로그인, 클라우드 동기화, AI 분석 등 서버 기능이 추가된다면 별도의 개인정보 처리방침과 사용자 동의 절차가 필요합니다.
